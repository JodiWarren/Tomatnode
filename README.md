Tomatnode
=========

Pomodoro timer in node.js

start with
    node tomatnode.js <pomodoro time> <short break time> <long break time>

example
    node tomatnode.js 25 5 15
    // pomodoros of 25 minutes, with short breaks of 5 minutes and long breaks of 15 minutes
